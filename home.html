What is our project?

Our final project is a website and game dedicated to educating viewers and players on political views,
as well as aligning their views with a 2020 Presidential Candidate.
We hope to have created an unbiased, informative source of education and information.
The website provides information on the project, download for the game, and information on the Presidential Candidates.
The game is setup as a visual novel, but is geared more towards introducing differing sides on issues rather than creating a narrative.

How did we make our project?

We used HTML, CSS, and Javascript to create the website.
For the game, we used the program RenPy, which uses a language based on Python.
